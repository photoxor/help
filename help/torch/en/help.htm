Introduction

This document is for Photoxor Torch version 1.0.0 (1), which can be installed or updated for free from Google Play.

Photoxor Torch is a simple flashlight application using the device camera flash LED as light source.

For latest information, tips and tricks please visit the Photoxor Blog.

Navigation Drawer

The Navigation Drawer is a feature to quickly access application functionality. To open the drawer please press the top left area of the action bar (where you see the three horizontal lines, the Photoxor icon and the application name Photoxor Torch. The navigation drawer opens and you are able to scroll through the functions provided. Click on one of the functions and directly go to the screen.

Application Widget

When you install Photoxor Torch an application widget is installed as well. You can find it in the list of widgets for your device.

The widget provides

  • The Photoxor icon to quickly start the application.
  • The Torch icon switches the torch on / off. Please note the automatic switch-off time, which can be configured in Settings. This function is implemented to protect battery usage and avoid overheating of device. Default is a 30 seconds on time.

In case the torch function is not supported by the device, the torch button is not presented and the widget looks like this:

Unfortunately in that case you can't use this application.

Settings

General Settings

Settings allow specific configuration of your application installation.

Settings - Configuration

Screen

This setting provides you with configuration of application screens.

You are able to customize the screen background and select either the default background or change to a custom color using the color picker. It is recommended to use a light color to retain good visibility of the application screen information.

Settings - Torch Auto Switch-Off Time

This setting allows to configure the automatic switch off time for the Torch.

Either select one of the time intervals or choose 'keep on' to disable the automatic switch-off. In this case please ensure that you manually switch off the torch.

Settings - Share

This function allows you to share a link to this application with your friends. Depending on what applications are installed on your device, you can for example use email, Facebook, instant messaging, cloud storage.

The shared link opens Google Play to download and install the application.

Settings - Google Analytics

This application collects some usage patterns through Google Analytics. The information is used to identify the general usage pattern of the application, which helps use to continue improvements of features. No personal information is used; all information is aggregated across the whole user community.

With this option you are able to opt-out from this feature.
