Introduction

This document is for WIW - Where I Was - version x.x, which can be installed or updated for free from Google Play

WIW - Where I Was - is a tool to record locations visited. Simplest, just shake your device and a recording of your current location (GPS coordinates) will be taken. Or automatically, for example when the App detects that you stop at a place or park your vehicle and start walking.

For latest information, tips and tricks please visit the WIW Blog.

Permissions Required

Android 6.0 (Marshmallow) introduced a new permissions model (see here for some information). When you run the application on Android 6.0 or later, permission are granted by the user while the app is running. You are now able to deny particular permissions, and also revoke already granted permissions.

Please note that in case permissions are not granted or revoked, the application might not work correctly and is limited in functionality.

The permissions required are:

CAMERA

  • Take picture

MICROPHONE

  • Perform Audio recording

STORAGE

  • Store pictures and audio recordings

LOCATION

  • Location tracking
  • Advertising

Usage

When you start WIW the first time, above screen is shown and your are ready to take your first Recording. Just press the orange colored + button.

What is a WIW 'Recording'

What is a 'Recording'? It is like a special message where the location coordinates are the key information. WIW makes it easy to record where you were.

Where - Location Coordinates
The location of the recording, most accurate by using GPS.
When - Date and Time
The date and time the Recording was made.
What - Note
Make a note what happened at this place.
Picture
Take a picture of the place.
Audio recording
Record some audio message.

Navigation Drawer

The Navigation Drawer is convenient feature to quickly access application functionality. To open the drawer please press the top left area of the action bar (where you see the three horizontal lines, the WIW icon and the application name WIW. The navigation drawer opens and you are able to scroll through the functions provided. Click on one of the functions to activate.

Application Widget

When you install WIW an application widgets is installed as well. You can find it in the list of widgets for your device.

Please note that widgets are not available when you move the application to the SD card; this is a behaviour of Android.

This widget provides

  • The WIW icon to quickly start the application.
  • Create a new Recording.
  • Record an audio message.
  • Record a photo.

Notifications

You may receive notifications from time to time on new application features, tips & tricks, and interesting articles. The notification icon changes from to as an alert that new messages are available.

Notifications are organised in three groups:

CURRENT
All the newly received notifications, followed by already read ones.
ARCHIVED
Keep for future reference.
DELETED
Marked for deletion. They will be removed within about 5 days.
The following icons are used:
MORE ...
Press "MORE ..." or just click on the notification text to follow an embedded link to further information (if available).
Archive the notification.
Delete the notification.
Undo the deletion of the notification. The notification will be moved to the CURRENT group.

Settings - Screen

This setting allows to configure the background of the application screens.

A white background is best for readability, but you may configure your own preferred color.

Use the color picker to select your custom background tint. It is recommended to use a light color to retain good visibility of the application screen information.

Settings - Shake Activation

Recordings can be created by simply shaking the device. This setting allows you to configure activation using the 'shake' feature, and what level of 'shake' should trigger.

We recommend to start with 'medium' level, and if required adjust to 'light' or 'hard'.

Settings - Google Analytics

This application collects some usage patterns through Google Analytics. The information is used to identify the general usage pattern of the application, which helps use to continue improvements of features. No personal information is used; all information is aggregated across the whole user community.

With this option you are able to opt-out from this feature.
