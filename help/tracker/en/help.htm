Introduction

With Photoxor Tracker you are able to track and record you path, visualize the track together with your favourite locations on a map, and quickly see what's closest to you. Record audio notes to quickly store information on your way, take geotagged photographs with your device camera. Store your favourite locations in KML files and load them into Tracker. You are able to present them on the map, but also list them in order of distance from your current location.

Select your preferred location provider:

  • GPS for the most accurate tracking, but depending on your device may consume more power
  • Network cell information uses less power, but provides only approximate location
  • Passive uses no additional power, but receives location information only when another application on your device requests it; good for background tracking.

A number of in-app purchases are available to further extend capabilities.

Please note that the accuracy of tracking is dependent on the GPS capabilities of your device.

Are you are photographer, professional or enthusiast? Then have a look at Photoxor C1; it provides tracking and additionally photography calculations.

Get it for free from Google Play.

Startup Screen

The Startup Screen provides the visual entry point into the application. Icons give you a one-click access to most of the application features. Color coding is used to present functionality grouping.

Icon Arrangement

The startup screen presents a number of icons as shortcuts for access to functionality of the application. The default arrangement might not be the best for your particular usage, and therefore prefer to rearrange icons. To do this, press and hold on the icon you want to move. On this 'long press' an array of squares shows up (see image) and you are now able to drag the chosen icon to one of the squares. As soon as you end the drag, the icon will be placed on the target square, and the array of squares dissapears. In case you drag to a square already holding an icon, this icon will be moved away.

Please repeat this process for other icons you want to rearrange. Different arrangements can be performed for portrait and landscape screen views.

New versions of the application may introduce new icons. They are placed onto one of the free positions, and you are able to then rearrange these new icons.

Tracking

Use GPS tracking to log your movement path to a file or display on a map.

Three screens (tabs) are available: TRACKING, MAP, PLACES

TRACKING Screen

From the Startup screen press the Tracking icon to reach the tracking functionality.

The tracking screen offers two key functionalities:

Tracking
A location provider (such as GPS) is used to determine the current location coordinates, and often also other details such as altitude, speed and bearing of movement. This information is shown on the screen and updated when available.
For GPS coordinates - depending on your device - it may take some time until the first fix is locked. In some instances, in particular within buildings you might not receive a GPS fix at all.
The location provider can be changed in Settings, also while Logging is active.
Logging
The current location information is recorded. It can be stored to a file where both GPX and KML formats are supported, or shown on the MAP screen only.
When you switch on Logging, the Tracking functionality is automatically enabled as well. You are able to switch off Tracking while Logging continues to be active. This can be used to save energy for example while inside a building or stationary at a location for some time.

While Logging is active both audio recordings and photos (taken with device camera) can be associated with the current location. You might use this feature to quickly record information about a place. For such recordings a marker is shown on the map.

Audio Recording

Wile Logging is active you are able to record an audio message. This recording is geocoded with the current location. In case the track is presented on the map (MAP screen) then a marker icon in shape of a microphone is shown.

Within Settings you can change the recording format between 3GP, MPEG4, AMR. The recording file is stored in a folder for media files associated with the track. The name is shown during the audio recording dialog.

Finish Logging

When you finish logging a dialog appears to select what action to take. Use Cancel in case you switched off logging in error.

You can share the log file, for example upload to cloud storage, send via email, ...

Depending on your settings (see Settings) the log file together with audio recordings and photos will be packaged as a ZIP file.

Please note that in case no records (for example no GPS fix could be obtained yet) are logged when you stop, no data from that session are available for sharing. Therefore in such situation pressing Share brings up a file dialog. You can then either select an old recording for sharing or abort.

The folloing log file formats are supported:

GPX
The GPX file format records the path with timestamp, location coordinates, speed, course, elevation.
This format is suitable for input into tools such as Adobe Lightroom.
KML
The KML file format records the path with timestamp and location coordinates.

MAP Screen

The MAP screen is based on Google Maps. It allows you to display the recorded path, a previously logged path, or mark places (see PLACES screen).

To show the map, Internet connectivity is required. You may use Google Map to persist a map area for offline access.

The Logger button is used to present the currently logged path on the map. Special markers are shown for audio recordings and photos taken.

The Load Log File button allows to load an existing log file and present the path on the map.

Use the None button to clear all markers and path.

PLACES Screen

The PLACES screen is used to load places (Load KML button) of interest from KML files.

The list of loaded places can be sorted either alphabetically or by distance from current location (if current location is available).

Click on one place and the MAP screen is shown, positioned for this place.

Use the On Map button to put markers for all places in the list onto the map (does not automatically move to MAP screen!). You can do this for multiple places lists.

Settings

General Settings

Settings allow specific configuration of your application installation and allows to enable functionality extension (either by in-app purchase or the integrated rewards system).

Settings - Share

This function allows you to share a link to this application with your friends. Depending on what applications are installed on your device, you can for example use email, Facebook, instant messaging, cloud storage.

The shared link opens Google Play to download and install the application.

Settings - Purchases

This function allows you to purchase additional functionality using Google in-app payment.

A list of purchasable features is presented:

Symbols indicate the purchase status for each item:

This symbol indicates that the feature is for purchase.
This symbol indicates that you have purchased the feature already and is ready for use. Or it is currently active for a trial run.
This symbol alerts that there is a problem with the purchase.

Press the Reload button to refresh your feature purchases from Google Play; this might resolve the alert.

Click on the item to view a more detailed description of the feature and the purchase price is shown:

On this screen you can abort the purchase using Cancel.

Before purchasing you may want to trial the feature. Use the TryMe button and the functionality will be enabled for a limited time (usually 15 minutes). The trial can be performed up to 3 times. Please note that the trial ends either on expiry of the trial time or exit from the application.

Use the Purchase button to buy the feature. You will be redirected to Google Play where you are able to perform payment.

Settings - Rewards

Reward points can be earned by completing surveys and viewing other advertisement content.

After earning some points you can use them to activate otherwise chargable functionality for a limited time.

The Earn Points screen lists surveys and advertisement. Click on one of the active buttons and complete the flow to earn the shown number of points. As soon as you've earned points the total points held in your wallet will be updated.

Please note that some of the content may not be available, and the button remains inactive. As soon as content becomes available the associated button will become active.

Once you have earned some points, redeem them by subscribing to one or more additional application functionality. On selection of one of the application features available to the rewards system, a Reward Detail screen is shown. This screen provides you with a feature description and your current subscription status for this feature, as well as the number of points required for a 24 hours activiation.

The Cancel button can be used to move back to the list screen without making any changes.

Use the Subscribe button to subscribe to the application feature. Once subscribed, the feature will be activated as soon as sufficient points are in your Wallet. As long as you maintain sufficient points in your Wallet the application feature will be reactivated on expiry of the 24 hours activiation, and the number of points will be deducted accordingly.

You may unsubscribe from the application feature at any time using the Stop button. The subscription will be ended, and access to the feature remains active until expiry of the activation period. You may subscribe to the feature again when needed.

Settings - Screen

This screen provides you with configuration of the application screens.

You are able to customise the screen background and select either the default background or change to a custom color using the color picker. It is recommeded to use a light color to retain good visibility of the application screen information.

Settings - Google Analytics

This application collects some usage patterns through Google Analytics. The information is used to identify the general usage pattern of the application, which helps use to continue improvements of features. No personal information is used; all information is aggregated across the whole user community.

With this option you are able to opt-out from this feature.
