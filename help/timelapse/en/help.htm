Introduction

This document is for Photoxor Timelapse version 1.0, which can be installed or updated for free from Google Play

Photoxor Timelapse provides time-lapse calculations to determine the capture parameters for a time-lapse video clip.

It also has a simple flashlight application using the device camera flash LED as light source.

An application widget allows quick access to the calculations and also to the Torch function.

A number of in-app purchases are available to further extend capabilities.

For latest information, tips and tricks please visit the Photoxor Blog.

Home Screen

Icon Arrangement

The Home Screen presents a number of icons as shortcuts for access to functionality of the application. The default arrangement might not be the best for your particular usage, and therefore prefer to rearrange icons. To do this, press and hold on the icon you want to move. On this 'long press' an array of squares shows up (see image) and you are now able to drag the chosen icon to one of the squares. As soon as you end the drag, the icon will be placed on the target square, and the array of squares disappears. In case you drag to a square already holding an icon, this icon will be moved away.

Please repeat this process for other icons you want to rearrange. Separate arrangements can be performed for portrait and landscape screen views.

New versions of the application may introduce new icons. They are placed onto one of the free positions, and you are able to then rearrange these new icons.

Navigation Drawer

The Navigation Drawer is an additional feature to quickly access application functionality. To open the drawer please press the top left area of the action bar (where you see the three horizontal lines, the Photoxor icon and the application name Photoxor C1. The navigation drawer opens and you are able to scroll through the functions provided. Click on one of the functions and directly go to the screen.

Please note that the Torch function is shown only when supported by your device.

Home Screen Configuration

For each function the Navigation Drawer shows a STAR icon, which allows to either place this function on the start screen (start in yellow) or hide it (start in grey). Now you are able to configure your individual start screen.

Don't want to show the start screen at all?

Then on the Navigation Drawer just select exactly one function, and the corresponding screen replaces your start screen.

Tools

Timelapse Calculations

"Time-lapse photography is a technique whereby the frequency at which film frames are captured (the frame rate) is much lower than that used to view the sequence. When played at normal speed, time appears to be moving faster and thus lapsing. For example, an image of a scene may be captured once every second, then played back at 30 frames per second; the result is an apparent 30 times speed increase." (Wikipedia)

The new time-lapses calculation tool helps to determine the correct camera recording settings for creation of a time-lapse clip.The calculation is performed in a few simple steps:

  1. First, define the length of the expected clip and the frame rate (frames per second - FPS). The tool now calculates the number of frames (images) required.
  2. Following define how the capture is done. For time-lapse the capture duration is much longer than the playback. Once the duration is defined, the tool calculates the required recording interval.
  3. Optionally you might want to set the anticipated start date/time, and get the finishing time.
  4. Make sure your camera memory is sufficient to hold the captured images. Enter the approximate image size and get the amount of storage needed.
  5. Now setup your camera, program the interval timer with the shooting interval and number of images required.

Time-lapse Pro

The "Time-lapse Pro" extension enhances the time-lapse tool features.

Duration and Interval values are presented as days, hours, minutes, seconds and tenth of seconds.

Graphical Dials

Keyboard entry on a small device is cumbersome? Then try the dial type data entry for duration and intervals. It allows to precisely enter in days, hours, minutes, seconds, down to a tenth of a second.

Go to settings and change the "Duration / Interval entry" to "Dial picker".

Custom defined Frame rates (FPS)

In addition to the provided standard frame rates you are able to define your own playback frame rates for a most accurate calculation.

Settings

General Settings

Settings allow specific configuration of your application installation and allows to enable functionality extension (either by in-app purchase or the integrated rewards system).

Settings - Definition Lists

The user is able to define custom lists of camera, filter and print definitions.

List Features

The definition lists do provide the following common features:

Add ...
The Add button allows to create a new definition. A dialog is presented for entry of specific values. All definitions use a name and optional make for identification.
Scrolling
For scrolling of the list touch and move on the right side (outside of the icon). This function is only available in case not all of the items within the list can be shown.
Export
Use the Export button to export all or some of the definitions to a XML file. A dialog allows the selection of the definitions to be exported.
The export file must have a suffix of .xml or .XML, otherwise .xml will be appended automatically.
Import
Use the Import button to import definitions from an XML file. Usually this file would be the result of a previous export. You may use the Export / Import functionality
to transfer definitions between devices running the application.
The file must have a suffix of .xml or .XML. to show in the file selection. A dialog allows the selection of the definitions to be imported.
Replace: The original list is replaced by the imported definitions. Please note that the original definitions are permanently deleted, and cannot be recovered.
Add: The imported definitions are added to the original list.
Reorder
You might prefer to show the most often used definitions at the beginning of the list.
To rearrange an item, touch the icon on the left side and drag to a new location.
Delete Definition
To delete a definition, touch the line on the right side (outside of the gray area) and swipe to the right. The item will disappear from the list and transferred to the archive. Use the archive function in case of accidental deletion.
Archive
Deleted definitions are held in a temporary archive while the application is running. In case you accidentally delete a definition, use this archive to restore it.
Whenever definitions are stored in the archive, the icon is shown above the list, together with the number of items held. Click on this icon for a list of
archived definitions. From the list select the items you would like to recover. Once restored an item is removed from the archive.
Reset
You can reset this list to default values; please note that all own definitions and customizations are permanently lost.
